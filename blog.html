2. How do you select an element from the DOM?

We can select element by few methods:


2. getElementsByClassName(className) Selects all elements with a specific class.

3. getElementsByTagName(tagName) Selects all elements with a specific tag name.

4. querySelector(selector) Selects the first element that matches a CSS selector.

3. What is event delegation in the context of the DOM, and why is it useful?

Definition: Event delegation is a technique in the DOM where a single event listener is added to a parent element to handle events on its child elements, even if those child elements are added dynamically. Instead of attaching an event listener to every individual child element, you attach it to the parent and take advantage of event bubbling to catch events from child elements.


There are many Event Delegation, such as click, mouseover, keyup, keydown. Each has different functionality and is used wide variant. We can declare a function and use event delegation to each element to do a certain work event and bring up change to certain part within DOM.

4. How do you manipulate an element's attributes and styles using the DOM?

By using getAttribute() and setAttribute() we can target and style or change any attribute of any specific element.


1. getAttribute(): By using this method we can target an attribute of an element to make changes later.

2. setAttribute(): By using this method we can style and make changes to the attribute that we targeted by using getAttribute().


Conclusion: To manipulate an element's attributes and styles in the DOM, you can use methods like getAttribute(), setAttribute(), or access properties directly. For styles, use the style property to apply inline CSS or modify classes with classList methods like add(), remove(), or toggle(). This gives you full control over how elements look and behave on a web page.
